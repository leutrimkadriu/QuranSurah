# 🕌 Quran Surah Explorer

A modern Islamic web application built with **HTML, CSS, and JavaScript** that allows users to explore Quran Surahs, listen to complete recitations, and discover information about the holy cities of **Makkah and Madinah**.

## ✨ Features

### 📖 Quran Explorer

* Browse Quran Surahs with a beautiful and responsive interface.
* Click on any Surah to open detailed information.
* Listen to the complete Surah recitation directly from the website.
* Easy navigation and smooth user experience.

### 🎧 Audio Quran Player

* Play Surah recitations.
* Control Quran audio directly from the application.
* Listen while exploring other sections.

### 🕋 Explore Makkah

* Dedicated page for Makkah.
* Search and explore important places.
* View images and information about Islamic landmarks.

### 🕌 Explore Madinah

* Dedicated page for Madinah.
* Discover historical and religious locations.
* Explore beautiful places connected with Islamic history.

### 🌙 Dark / Light Theme

* Modern theme switcher.
* Toggle between dark and light modes.

### ⭐ Interactive UI

* Beautiful cards and animations.
* Modal windows for detailed views.
* Favorite and interactive elements.
* Smooth scrolling experience.
